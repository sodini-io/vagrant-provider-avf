Vagrant, native on Apple Silicon.

A lean Vagrant provider for Apple Silicon Macs built on Apple Virtualization.framework, with curated ARM64 Linux boxes and a real headless workflow.

View on GitHub Install the Gem Jump to Quickstart

Quick install

vagrant plugin install vagrant-provider-avf
vagrant box add sodini-io/ubuntu-24.04-arm64 --provider avf --box-version 0.1.0
vagrant up --provider avf

Native runtime

Uses Apple Virtualization.framework through a signed local helper instead of piling a new VM stack on top.

Focused scope

Linux only, headless only, Apple Silicon only. Small enough to understand. Boring enough to trust.

Real workflow

up, halt, destroy, ssh, honest ssh_info, and read/write Linux shared folders.

Quickstart

From install to login in three steps

01

Install the provider

vagrant plugin install vagrant-provider-avf
02

Add a box

vagrant box add sodini-io/ubuntu-24.04-arm64 --provider avf --box-version 0.1.0
03

Bring it up

vagrant up --provider avf
vagrant ssh

Minimal Ubuntu

Vagrant.configure("2") do |config|
  config.vm.box = "sodini-io/ubuntu-24.04-arm64"
  config.vm.box_check_update = false

  config.vm.provider :avf do |avf|
    avf.cpus = 2
    avf.memory_mb = 2048
    avf.disk_gb = 8
    avf.headless = true
  end
end

Shared folders and tuning

Vagrant.configure("2") do |config|
  config.vm.box = "sodini-io/ubuntu-24.04-arm64"
  config.vm.box_check_update = false

  config.vm.synced_folder ".", "/vagrant"
  config.vm.synced_folder "examples", "/home/vagrant/examples", type: :avf_virtiofs

  config.vm.provider :avf do |avf|
    avf.guest = :linux
    avf.cpus = 4
    avf.memory_mb = 4096
    avf.disk_gb = 16
    avf.headless = true
  end
end
